블로그에서 자주 쓰인 문법을 중심으로 포스팅하고자 한다.


## 1. 링크 걸기
<br>

링크 앞뒤에 <,>를 붙이면 링크가 걸린다.
```python
<https://www.naver.com>
```
<https://www.naver.com>
<br>

글자에 링크를 걸고 싶다면 대괄호[]와 소괄호()를 이용해준다. 

```python
[네이버](https://www.naver.com)
```
[네이버](https://www.naver.com)
<br><br>

## 2. 코드 표시

`인라인 코드`는 어떻게 할까요?

```python
`이렇게` 걸어주면 됩니다.
```

코드를 길게 쓰고 싶다면

\```언어명

...코드...

```
이런식으로 작성을 해주면 됩니다.


```python

int i
for i in range(1, 10):
  print(i)

```
